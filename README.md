## About the Site
- Static site written entirely in Markdown
- Hosted via GitHub Pages
- Edited locally using PyCharm, built by Jekyll and kramdown

## Repository Structure
- `index.md` – Website home page
- `README.md` - Splash screen on the github repo
- `pages/` – Main site content
- `assets/` – Images, documents, and styling (main.scss)

## Editing & Preview
- Edit Markdown files locally
- Push to `main` branch to publish
- GitHub Pages builds automatically using kramdown (specified in config.yml)
- Supports MathJax
